## Passport Token Management

### Create Token
- [ ] Click "Yeni Pasaport Linki Oluştur" — form appears
- [ ] Fill label (optional) — works
- [ ] Fill expiry date (optional) — works
- [ ] Toggle visibility settings — checkboxes work
- [ ] Click "Oluştur" — token created, page reloads
- [ ] Token appears in token list

### Toggle Token
- [ ] Click toggle icon on active token — token deactivates
- [ ] Click toggle icon on inactive token — token reactivates
- [ ] Deactivated token shows inactive visual state

### Delete Token
- [ ] Click delete icon — confirmation dialog appears
- [ ] Confirm delete — token removed, page reloads

### Copy Link
- [ ] Click "Linki Kopyala" — link copied to clipboard
- [ ] "Kopyalandı!" feedback shown briefly
- [ ] Pasted URL is correct format: `{origin}/p/{token}`

### Preview
- [ ] Click "Önizle" — opens public passport page in new tab

### QR Code
- [ ] QR code displayed for active, non-expired tokens
- [ ] QR code encodes correct URL
- [ ] QR code not shown for inactive or expired tokens
