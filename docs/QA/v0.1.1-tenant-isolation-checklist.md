2. Her profilde farklı bir hesapla giriş yapın

## Test 1: Müşteri İzolasyonu

### 1.1 Hazırlık
- [ ] Workshop A ile Müşteri A1 oluşturun (örnek: Ahmet Yılmaz, 0555 111 1111)
- [ ] Workshop B ile Müşteri B1 oluşturun (örnek: Mehmet Demir, 0555 222 2222)
- [ ] Her iki workshop'ta da yalnızca kendi müşterisi görünür

### 1.2 Cross-Workshop Erişim Denemeleri
- [ ] Workshop A, tarayıcıdan `/app/customers/[B1_Müşteri_ID]` adresini açar → 404 sayfası
- [ ] Workshop B, tarayıcıdan `/app/customers/[A1_Müşteri_ID]` adresini açar → 404 sayfası

### 1.3 API Cross-Workshop
- [ ] Workshop A, konsoldan Workshop B'nin müşterisini güncellemeye çalışır → hata yanıtı

---

## Test 2: Araç İzolasyonu

### 2.1 Hazırlık
- [ ] Workshop A, Müşteri A1'e bağlı Araç A1 oluşturun (34 TEST 01)
- [ ] Workshop B, Müşteri B1'e bağlı Araç B1 oluşturun (34 TEST 02)

### 2.2 Cross-Workshop
- [ ] Workshop A, Araç B1 oluşturmak için Müşteri B1'i seçemez (dropdown'da görünmez)
- [ ] Workshop A, API üzerinden Müşteri B1 ID'si ile araç oluşturmaya çalışır → hata
