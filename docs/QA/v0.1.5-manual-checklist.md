### QA Sırasında Bulunan ve Düzeltilen Sorunlar

1. **`source` filtresi bozuktu** — `customer-list.tsx:84` her zaman `false` dönüyordu, filtre hiç çalışmıyordu. Düzeltildi, kaynak dropdown'ı eklendi.
2. **Müşteri listesinde `Sil` butonu yoktu** — Checklist "İşlem" kolonunda Sil bekliyordu, sadece detay sayfasında vardı. `customer-list-with-delete.tsx` client wrapper'ı eklendi, list sayfasında onaylı sil akışı aktif.
3. **Seed idempotent değildi** — İkinci çalıştırmada `Unique constraint failed on email` hatası. Workshop/user/customer kayıtları için "find-first, create if missing" mantığına geçildi.
4. **Source enum validation** — Boş string enum validation'da hata veriyordu, `.or(z.literal(""))` ile boş string kabul edildi.
