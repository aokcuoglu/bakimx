# v0.2.1 — Araçlar Modülü UX Alignment

**Sürüm:** v0.2.1
**Tema:** Araç listesi, detay, arama ve yönetim profesyonelleştirmesi

---

## Yeni Özellikler

### 1. Araç Listesi UX Yenileme (`/app/vehicles`)

- **Sayfa başlığı:** "Araçlar" + breadcrumb "Ana Panel / Araçlar"
- **Birincil CTA:** "+ Yeni Araç" butonu
- **İkincil CTA:** "Plaka Tara" butonu (placeholder — plaka tanıma entegrasyonu yakında)
  - Yardımcı metin: "Plaka tanıma entegrasyonu yakında."
- **Filtreler:**
  - Ana arama: "Plaka, marka, model veya müşteri ara..."
  - Araç tipi filtresi: Tüm Tipler, Binek, Hafif Ticari, Ağır Vasıta, Motosiklet, Diğer
  - Marka filtresi (mevcut araçlardan otomatik derlenir)
  - Filtreleri temizle butonu
- **Masaüstü tablosu:** Plaka (PlateBadge), Araç (marka/model), Müşteri (link), Kilometre, İş Emri sayısı, Son İşlem tarihi, Kayıt Tarihi, İşlem (Görüntüle, Düzenle, Sil)
- **Mobil kartları:** PlateBadge, brand/model, müşteri, kilometre, iş emri sayısı, son işlem, aksiyonlar
- **Boş durumlar:** "Henüz araç kaydı yok" / "Aramanızla eşleşen araç bulunamadı"
- **Tenant izolasyonu:** Tüm listeleme/arama sorguları `workshopId` ile kapsamlı

### 2. Yeni Araç Sayfası (`/app/vehicles/new`)

- **İki sütunlu yerleşim (masaüstü):** Ana form kartı + sağ ipucu paneli
- **Form bölümleri:**
  - A) **Müşteri Bağlantısı** — müşteri seçici, yeni müşteri linki
  - B) **Araç Bilgileri** — Plaka, Marka, Model (zorunlu); Araç Tipi, Model Yılı, Kilometre, Renk (opsiyonel)
  - C) **Teknik Bilgiler** — Şase No (VIN), şase teyit checkbox, Motor No, Yakıt Tipi, Şanzıman
  - D) **Notlar** — Araç notu
- **"Ruhsattan Oku" placeholder:** "Ruhsattan şase okuma özelliği yakında."
- **Mobil sticky alt aksiyon çubuğu:** Kaydet / İptal
- **Validasyon:** Türkçe hata mesajları, plaka normalizasyonu

### 3. Araç Detay Sayfası (`/app/vehicles/[id]`)

- **Başlık bölgesi:** PlateBadge (büyük), marka/model, araç tipi/yıl/renk/yakıt/şanzıman, hızlı aksiyonlar
- **Bölümler:**
  - **Araç Özeti:** Grid kartlarda tüm araç bilgileri, şase teyit durumu, kayıt/güncelleme tarihleri
  - **Müşteri Bilgisi:** Müşteri adı, telefon, e-posta, müşteri detayına link
  - **İş Emri Geçmişi:** İş emri no, durum/ödeme rozetleri, şikayet, toplam, tahmini teslim
  - **Araç Kabul Geçmişi:** Kabul durumu, onay durumu, şikayet, tarih
  - **Hasar Geçmişi:** Tüm kabullerden hasar kayıtları, bölge, tip, şiddet, tarih, kabul detayına link
  - **Fotoğraf Geçmişi:** Tüm kabullerden fotoğraflar, thumbnail grid, tip etiketi, tarih
  - **Notlar:** Araç notları (varsa)
- **Yan panel:**
  - Araç Durumu kartı (işlem var/yok, açık iş emri, onay bekliyor, hasar kaydı, fotoğraf)
  - Düzenle ve Yeni Kabul butonları
- **Mobil uyum:** Tek sütun, kart bazlı bölümler, touch-friendly butonlar

### 4. Araç Düzenleme Sayfası (`/app/vehicles/[id]/edit`)

- Yeni araç formu ile aynı alanlar
- Tenant sahiplik kontrolü
- Türkçe validasyon mesajları
- Güvenli iptal/geri aksiyonu (detay sayfasına döner)

### 5. Arama İyileştirmeleri

- Plaka, marka, model, müşteri adı/şirket, telefon ile arama
- Araç tipi filtresi
- Marka filtresi (dinamik, mevcut veriden)
- Tüm sorgular `workshopId` ile kapsamlı
- Harici arama motoru eklenmedi

### 6. PlateBadge Bileşeni (`src/components/app/plate-badge.tsx`)

- Tutarlı plaka stili: lacivert arka plan, beyaz monospace metin
- Araç listesi, detay, iş emirleri, müşteri detayında kullanılır
- Erişilebilir, responsive, abartısız

### 7. Şema Genişletmeleri

- `Vehicle` modeline eklendi (tümü opsiyonel, geriye dönük uyumlu):
  - `color` — Araç rengi
  - `engineNo` — Motor numarası
  - `fuelType` — Yakıt tipi
  - `transmission` — Şanzıman tipi
  - `notes` — Araç notu

- `vehicleUpdateSchema` eklendi (Zod validasyonu)
- `VEHICLE_TYPES`, `VEHICLE_FUEL_TYPES`, `VEHICLE_TRANSMISSIONS` sabitleri

### 8. API Rotaları

- `GET /api/vehicles/[id]` — Tek araç getir (workshop-scoped)
- `PUT /api/vehicles/[id]` — Araç güncelle (FormData)
- `DELETE /api/vehicles/[id]` — Araç sil (bağlı kayıt varsa güvenli hata)

### 9. Placeholder Özellikler

- **"Plaka Tara"** butonu yalnızca UI placeholder (gerçek plaka tanıma/OCR değil)
- **"Ruhsattan Oku"** butonu yalnızca UI placeholder (gerçek OCR/VIN çıkarımı değil)
- Her iki özellik için açık yardımcı metin gösterilir

---

## Tenant İzolasyonu

- Tüm araç sorguları `workshopId` ile kapsamlı
- Müşteri seçici yalnızca mevcut workshop müşterilerini gösterir
- İş emri ve kabul geçmişi workshop-scoped
- Hasar/fotoğraf geçmişi workshop-scoped
- Doğrudan URL erişiminde workshopId doğrulaması
- Silme işlemi bağlı kayıt varsa güvenli hata döndürür

---

## Regresyon Güvenliği

Tüm önceki rotalar çalışmaya devam eder:
- Landing page (`/`), login (`/login`), register (`/register`)
- Dashboard (`/app`), iş emirleri (`/app/orders/*`)
- Müşteriler (`/app/customers/*`), araç kabulleri (`/app/intakes/*`)
- Fotoğraf yükleme/depolama, hasar işaretleme
- Mock SMS onayı, public output (`/s/[token]`)
- PDF/print, WhatsApp paylaşımı
- Hiçbir Docker dosyası eklenmedi

---

## Sınırlamalar (v0.2.1)

- Gerçek plaka tanıma / OCR entegrasyonu yok (placeholder)
- Gerçek VIN çıkarımı yok (placeholder)
- Araç sağlık puanlaması yok (eklenmedi, misleading olabilir)
- Predictive bakım yok
- Renk, motor no, yakıt tipi, şanzıman alanları yeni eklendi, mevcut kayıtlarda boş
- Araç silme sadece bağlı kabul kaydı olmayan araçlar için mümkün
- Fotoğraf geçmişi thumbnail'leri storage provider'a bağlı
- Docker konteyner desteği yok (lokal bun/npm)
