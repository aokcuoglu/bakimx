# BakimX v0.1.0 — Vehicle Intake MVP

## Summary
BakimX v0.1.0 introduces the first functional vehicle intake MVP for auto repair shops. Workshops can now register/login, manage workshop profile information, create customers and vehicles, start vehicle intake records, use a guided photo checklist, mark vehicle damages, run a mock SMS approval flow and generate tokenized customer-facing output links.

## Included
- Authentication foundation
- Workshop profile management
- Customer records
- Customer detail page
- Vehicle records
- Mobile-first protected app dashboard
- Vehicle intake wizard
- Guided vehicle photo checklist
- Correct photo type selection flow
- 2D SVG vehicle damage marking
- Mock SMS OTP approval flow
- Intake-based OTP verification
- Separate approval/consent checkboxes
- Tokenized public customer output page
- Basic service order structure
- Part/labor service order foundation
- Audit log foundation
- Prisma/PostgreSQL-compatible schema
- Build-safe Prisma fallback when DATABASE_URL is unset
- README and .env.example documentation
- Local development with bun/npm
- No Docker local workflow
