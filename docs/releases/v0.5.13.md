# BakımX v0.5.13 — Billing/Satın Alma Akışı & Release Pipeline

**Release Date:** June 2026
**Type:** Özellik (büyük) + dağıtım altyapısı
**Migration:** Evet (`billing_orders`)

## Overview
İki büyük blok: (1) uçtan uca **abonelik/satın alma akışı**, (2) **dev/staging/main release pipeline**. Ayrıca birleşik müşteri+araç seçici ve kabulden otomatik iş emri üretimi geldi.

## Yeni — Billing / satın alma
- Public **/fiyatlar** ve **/satın-al** sayfaları; KDV dahil kuruş bazlı fiyatlandırma.
- **In-app checkout** sihirbazı + `createBillingOrder` server action; `BillingOrder` modeli + workshop dönem alanları.
- Public **POST /api/checkout** — pending-payment işletme + sipariş oluşturur (e-posta tekrarı 409, referans çakışmasında retry).
- Dönem matematiği (`addPeriod`, `periodStartFrom`), dönem sonu kilidi (`subscription_expired`), ödeme sağlayıcı seam'i (manuel havale).
- **Admin billing**: gelen kutusu, sipariş onay/iptal (idempotent), abonelikler ve gelir paneli; orta-dönem yükseltmede prorate.
- HTML abonelik **makbuzu** (receipt) rotası; `/billing` üzerinde yenileme geri sayımı ve pending-payment afişi.

## Yeni — İş emri & seçici
- Birleşik **müşteri+araç arama API'si** ve searchable picker; inline "müşteri+araç oluştur" modalı.
- Kabul (intake) oluşturulunca **otomatik iş emri** üretimi (`createServiceOrderForIntake`).
- Araç sahibi değiştirme (`changeVehicleOwnerAction`).

## Yeni — Release pipeline
- `dev → staging (otomatik) → main (tag) → prod` akışı; **RELEASE.md**.
- Staging stack (compose + workflow + env şablonu + setup rehberi).
- Prod app yeniden oluşturulmadan önce **tek-seferlik prisma migrate gate**.
- Ortam-izole edilebilir session cookie adı/domaini.
- 63 dosya (+6774 / −518).
