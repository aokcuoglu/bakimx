# BakimX v0.2.4 — Stok / Parçalar Foundation

## Summary
BakimX v0.2.4 introduces the foundation for workshop parts and stock management. Workshops can now maintain a basic parts catalog, track stock quantities, define critical stock thresholds, record manual stock movements and select catalog parts in work orders and quotes.

## Included
- Stok / Parçalar module foundation
- Parts list, create, detail and edit flows
- Part search and filtering
- Stock status badges
- Critical stock threshold
- Out-of-stock visibility
- Manual stock movement foundation
- Basic supplier text fields
- Add catalog part to work order
- Add catalog part to quote
- Dashboard critical stock widget
- README/release notes/QA checklist updates
- Local development with bun/npm
- No Docker local workflow

## Placeholder / foundation-only features
- Supplier fields are basic text fields only
- Supplier API integration is not implemented
- Barcode scanner is not implemented
- Automatic stock deduction is not enabled unless explicitly implemented safely
- Stock movement is operational tracking only, not accounting

## Not included
- Docker
- Supplier API integration
- E-commerce product feed logic
- Purchase invoice / supplier invoice
- Payment/billing/invoice/e-archive module
- Full accounting
- Full warehouse module
- Multi-branch enterprise management

## Security
- All queries filtered by workshopId
- PartStockItem and StockMovement models require workshopId
- Direct URL access to other workshop parts is blocked
- Work order/quote part selections are workshopId scoped
- Stock movements use transactions
