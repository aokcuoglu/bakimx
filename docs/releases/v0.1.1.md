# BakimX v0.1.1 — MVP Hardening & UX Polish

## Summary
BakimX v0.1.1 hardens and polishes the Vehicle Intake MVP introduced in v0.1.0. This release focuses on tenant isolation, auth/session reliability, mobile usability, intake flow clarity, mock SMS robustness, public output polish, service order usability and documentation.

## Included
- Tenant isolation hardening across protected data access
- Auth/session polish
- Improved protected app mobile UX
- Better page headers, empty states and error states
- Intake wizard flow improvements
- Guided photo checklist polish
- Required/optional photo clarity
- 2D damage marking UX improvements
- Mock SMS OTP approval hardening
- Public token output page polish
- Basic service order polish
- Safer validation and Turkish error messages
- Audit log consistency improvements
- README updates
- v0.1.1 manual QA checklist
- Local development with bun/npm
- No Docker local workflow

## Not included
- Docker
- Real SMS provider
- Real OCR/VIN extraction
- Real image storage provider
- WhatsApp Business API
- Payment/billing
- Advanced reporting
- Labor price memory
- Multi-branch enterprise management

## Kapsam (Türkçe)

v0.1.1, v0.1.0 MVP'nin kararlılık, güvenlik ve UX iyileştirmelerini içerir. Yeni büyük modüller eklenmez.

### Tenant İzolasyonu Güçlendirmesi
- Tüm server action ve API route'lar workshopId kapsamı açısından denetlendi
- `getCurrentUserWithWorkshop()` yardımcı fonksiyonu eklendi
- `assertWorkshopAccess()` yardımcı fonksiyonu eklendi
- `isAuthenticated()` yardımcı fonksiyonu eklendi
- Middleware, API route'larına 401 yanıtı ekledi
- Müşteri/araç/kabul detay sayfaları workshopId sahiplik kontrolü yapıyor
- Cross-workshop veri erişimi engellendi

### Auth/Session Güçlendirmesi
- loginAction: E-posta trim ve lowercase normalizasyonu eklendi
- loginAction: Workshop varlığı kontrolü eklendi
- registerAction: E-posta trim ve lowercase normalizasyonu eklendi
- Hata mesajları daha açıklayıcı hale getirildi
- Oturum açmış kullanıcılar /login ve /register sayfalarından yönlendiriliyor
- Cookie güvenlik ayarları: httpOnly, secure (production), sameSite: lax
- SESSION_SECRET production hardening
- Logout güvenilirliği artırıldı

### Mobil UX İyileştirmeleri
- Tüm korumalı sayfalarda iyileştirilmiş mobil aralıklar
- Dokunmatik dostu butonlar (minimum h-12 boyut)
- Mobil alt navigasyon geliştirildi
- Mobil sidebar kapatma (X) butonu eklendi
- Boş durumlar daha açıklayıcı
- Yükleme/hata durumları daha belirgin
- Durum rozetleri daha okunabilir
- "Geri" navigasyonu geliştirildi
- Kart/liste tabanlı mobil görünüm
- Header'da "Giriş Yap" linki

### Kabul Sihirbazı İyileştirmeleri
- Progress bar görsel ilerleme göstergesi eklendi
- Adım numarası ve başlığı gösterimi eklendi
- Tamamlanan adımlar yeşil ile işaretleniyor
- Zorunlu alan validasyonları
- Müşteri şikayeti zorunlu alan kontrolü
- Onay adımında onay metni checkbox'ları eklendi
- Daha iyi hata durumu gösterimi
- Durum geçişleri: draft → waiting_approval → approved → in_progress → ready_for_delivery → delivered
- İptal (cancelled) durumu eklendi

### Fotoğraf Kontrol Listesi İyileştirmeleri
