# BakimX v0.4.2 — Operational Analytics

## Overview

BakimX v0.4.2 adds **Operational Analytics** — a comprehensive analytics dashboard providing actionable workshop intelligence through rule-based recommendations and data-driven metrics.

**No AI/LLM dependency. No forecasting models. Uses existing data only.**

## Route

`/app/analytics` — Operational Analytics page

## Features

### 1. Operations Health Dashboard

- Health score (0-100) based on overdue jobs, critical stock, and unpaid work orders
- Score ranges: 80+ (Good/İyi), 50-79 (Medium/Orta), below 50 (Critical/Kritik)
- Visual ring chart with color-coded scoring
- Metric cards: active jobs, delayed jobs, waiting approvals, critical stock, open receivables

### 2. Delayed Jobs

- Lists overdue work orders with days delayed
- Shows technician, customer, vehicle, and order total
- Status badges from existing ORDER_STATUS constants

### 3. Technician Analytics

- Average completion time per technician
- Active workload (current active jobs)
- Completed jobs count
- Rankings: fastest technicians, busiest technicians
- Performance summary table

### 4. Customer Analytics

- Total customers, new this month
- Repeat customer rate (%)
- Inactive customers (90+ days without an order)
- Top 10 high-value customers by spending

### 5. Parts Analytics

- Total parts, stock value, critical stock count, out-of-stock count
- Most consumed parts (from stock movements)
- Stock risk list with critical/tükenen status

### 6. Revenue Analytics

- Total collected, open receivable
- Average ticket size (per completed order)
- Revenue per work order
- 6-month collection trend bar chart

### 7. Service Analytics

- Most common complaints (from customerComplaint)
- Most common repairs (from ServiceOrderItem names)
- Most common labor items with revenue

### 8. Recommendation Engine

Rule-based only. Generates actionable alerts like:

- "5 iş emri 3 günden fazla gecikmiş durumda."
- "12 parça kritik stok seviyesinin altında."
- "Müşteri geri dönüş oranı %45 — iyi seviye."

Categories: delayed, stock, revenue, approvals, customers, health

### 9. Dashboard Integration

- Operational Alerts widget added to main dashboard (`/app`)
- Shows top 3 recommendations with link to full analytics page
- Sidebar navigation updated with "Operasyonel Analiz" link

```
src/components/app/analytics/          — Analytics UI components
  health-score-card.tsx

  technician-analytics.tsx
  customer-analytics.tsx
  parts-analytics.tsx
  revenue-analytics.tsx
  service-analytics.tsx
  recommendations-list.tsx

```

## Data Flow

- All queries use `prisma` directly (no new API routes)
- Workshop-scoped (`workshopId` filter)
- Server component fetches all data in parallel via `Promise.all`
- No new Prisma schema changes required
- No Docker, no database changes
