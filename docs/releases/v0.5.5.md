# BakımX v0.5.5 — Self-Hosted Yığına Geçiş

**Release Date:** June 2026
**Type:** Altyapı / mimari geçiş (yeni ürün özelliği yok)
**Migration:** Yok (şema değişmedi)

## Overview
Supabase bağımlılığından çıkıp tamamen kendi kendine barındırılan (self-hosted) bir yığına geçildi. Veritabanı doğrudan PostgreSQL'e (`pg` adapter ile Prisma), depolama ise S3-uyumlu sağlayıcıya (yerelde MinIO, üretimde Cloudflare R2) taşındı. Bu, Contabo VPS üzerinde tam kontrol ve maliyet öngörülebilirliği sağlar.

## Değişiklikler
- Supabase auth/storage/db bağımlılıkları kaldırıldı; yerine `pg` + S3 istemcisi.
- Depolama soyutlaması S3-uyumlu sağlayıcı üzerinden (MinIO/R2/mock).
- Yerel altyapı `docker-compose.local.yml` ile (PostgreSQL + MinIO).
- 16 dosya güncellendi (+520 / −105).
